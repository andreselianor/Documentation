Clase 'UpFile'

Main Menu

- documentación web -

Esta clase contiene toda la información que necesitamos del archivo que deseamos copiar, y la transforma en un objeto de tipo 'upFile'. Es más funcional trabajar con un objeto que nosotros mismos hemos creado a partir de la información original.


[#] upFile ()
Crea un archivo 'upFile' con atributos vacios.


[#] upFile (upfilePath, filePath)
Crea un archivo 'upFile' con una referencia a sus direcciones y el contenido del archivo.


- file: byte[]
Contiene el array de bytes con el contenido del archivo.


- path : upFilePath
Contiene una referencia a la clase 'upFilePath' donde se almacenan las direcciones completas y relativas del archivo.


- date : DateTime
Almacena la fecha de creación del archivo.


- size : int
Almacena el tamaño en bytes del archivo.


- hash : int
Contiene el número hash del archivo.


- SHA256 : string
Contiene el archivo, codificado en una cadena de texto, según el algoritmo SHA256.
